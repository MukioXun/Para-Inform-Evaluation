## 1. 数据结构重构

### 三层嵌套结构
```
{
  "audio_id": "xxx",
  "file_path": "/path/to/audio.wav",
  "content_metadata": {
    "duration_seconds": 10.5,
    "sample_rate": 16000,
    "channels": 1,
    "format": "wav"
  },
  "acoustic_features": {
    "low_level": { ... },
    "embeddings": { ... },
    "high_level": { ... }
  }
}
```

### 删除的冗余字段
- `task`: 不再在每个结果中存储任务名
- `status`: 移除，错误通过 error 字段表示
- `timestamp`: 移除顶层时间戳

## 2. 新建模块

### 2.1 Low-Level 特征提取器
**文件**: `annotators/lowlevel/feature_extractor.py`

| 维度 | 特征 | 工具 |
|------|------|------|
| Spectral | MFCCs, Mel-stats, Centroid, Bandwidth, Rolloff, Flatness | librosa |
| Prosody | F0 (stats), Jitter, Shimmer, HNR | pyworld |
| Energy | RMS (stats), Loudness, Dynamic Range | librosa |
| Temporal | Duration, Speech_rate, Pause_ratio, Voiced_ratio | silero-vad |
| Timbre | Formants (F1-F3), Spectral_envelope, Harmonicity | LPC/librosa |

### 2.2 Embeddings 提取器
**文件**: `annotators/embeddings/embedding_extractor.py`

| 模型 | 维度 | 来源 |
|------|------|------|
| wav2vec2-base-960h | 768 | HuggingFace |
| HuBERT-base-ls960 | 768 | HuggingFace |
| CLAP-htsat-unfused | 512 | HuggingFace |

提取方式: Utterance-level, Mean Pooling

### 2.3 SAR 标注器 (SenseVoiceSmall)
**文件**: `annotators/sar/sensevoice_attribute.py`

- 模型路径: `/home/u2023112559/qix/Models/Models/SenseVoiceSmall`
- 输出: Gender, Age, Embedding
- 备用方案: 简化 ECAPA-TDNN 编码器
